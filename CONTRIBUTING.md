# Contributing to SCPL Official Libraries

Привет! Спасибо, что хочешь добавить свою библиотеку в официальный реестр SCPL.

## Требования к библиотеке

Чтобы я принял заявку, твой репозиторий должен содержать три обязательных файла:

### 1. `__init__.scpl`

Главный файл библиотеки. Именно он загружается через `import имя`.  
В нём должны быть все функции, которые ты хочешь экспортировать.

Пример:

```scpl
import path
Initialize-console

Console-print: (path-join 'folder' 'file.txt')

Close-console
```

### 2. `dependencies.json`

Список зависимостей (если они есть). Даже если зависимостей нет, файл должен существовать.

Пример пустого файла:

```json
{}
```

Пример с зависимостями:

```json
{
  "match": "latest",
  "guard": "latest"
}
```

### 3. `README.md`

Описание библиотеки: какие функции экспортирует, примеры использования, лицензия.

Минимальное содержимое:
- Название библиотеки
- Список экспортируемых функций с кратким описанием
- Один-два примера использования

## Как подать заявку

1. Форкни репозиторий с официальным списком библиотек (или создай Issue, если форка нет)
2. Добавь свою библиотеку в `libs_list.txt` в формате:

```text
название=https://github.com/твой-ник/твой-репозиторий.git
```

3. Отправь Pull Request или создай Issue с ссылкой на репозиторий

## Что будет дальше

Я проверю:
- Есть ли все три обязательных файла
- Корректно ли заполнен `dependencies.json`
- Работает ли `__init__.scpl` без ошибок
- Понятный ли README

Если всё хорошо — я добавлю библиотеку в официальный список.  
Если нет — напишу, что нужно исправить.
